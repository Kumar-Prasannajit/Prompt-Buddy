## ðŸ“¦ Installation (Developer Mode)

### 1. Add Icon Files
Create the following files in the extension root directory:
- `icon16.png`
- `icon48.png`
- `icon128.png`

> Tip: A prompt / edit / magic-wand style icon works well.

---

### 2. Load the Extension in Chrome
1. Open `chrome://extensions/`
2. Enable **Developer mode** (top-right toggle)
3. Click **Load unpacked**
4. Select the `promptBuddy` folder

---

### 3. Configure PromptBuddy
1. Click the PromptBuddy icon in the Chrome toolbar
2. Open **Settings**
3. Select your AI provider:
   - OpenAI
   - Gemini
   - Claude
4. Paste your API key
5. (Optional) Set tone and max words
6. Click **Save**

---

### 4. Use PromptBuddy
1. Open the extension
2. Paste or type a messy prompt
3. Click **Enhance Prompt**  
   or press **Ctrl / Cmd + Enter**
4. Copy or insert the improved prompt directly into your AI tool
